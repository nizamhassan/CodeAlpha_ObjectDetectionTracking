# 🎯 Object Detection and Tracking using YOLOv8 + SORT

## 📌 Project Overview

This project implements a real-time object detection and tracking system using **YOLOv8** for detection and **SORT (Simple Online and Realtime Tracking)** for tracking.

The system captures video input from a webcam, detects multiple objects in each frame, and assigns unique tracking IDs to each object.



## 🚀 Features

* Real-time video capture using OpenCV
* Object detection using YOLOv8
* Bounding box visualization
* Unique ID tracking using SORT
* Real-time performance

## 🛠️ Technologies Used
